# DS-2002 Data Project 1 — Medical Appointments Data Mart

## How to Run This Notebook

### Step 1 — Install required packages
```
pip install pandas pymongo sqlalchemy pymysql certifi
```

### Step 2 — Download the dataset
Download the CSV files from Kaggle and place them in the same folder as the Project 1.ipynb:
https://www.kaggle.com/datasets/carogonzalezgaltier/medical-appointment-scheduling-system

Required files:
- appointments.csv
- patients.csv
- slots.csv

### Step 3 — Update credentials in Cell 2
- Replace `uid` and `pwd` with your MySQL username and password
- Replace MongoDB credentials with your own Atlas credentials

### Step 4 — Run the notebook
Click **Kernel → Restart & Run All**

## Data Source
Medical Appointment Scheduling System dataset from Kaggle:
